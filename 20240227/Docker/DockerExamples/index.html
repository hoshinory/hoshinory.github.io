Docker Practice

1. Docker编译安装Python

1.1 Python编译所需必要

  • Python-3.11.7 源码
  • Openssl-3.2.0
  1. 必须手动编译安装openssl(yum install openssl 不生效), 否则python编译完成后, pip install package 会报SSH错误
  2. 版本选择, 在对应链接里下载最新的3.2.x源码即可

1.2 TensorRT-8.6 与CUDA 及 CUDANN

  • TensorRT-8.6.1.6
  • CUDA-12.2.2
  • CUDNN-8.8.1
  1. TensorRT与CUDA和CUDNN版本依赖非常强, 如果对应不上, 后续做TRT-ENGINE转换的时候会有很多莫名其妙的错误

1.3 打镜像

  • 使用docker pull nvidia/cuda:12.2.2-cudnn8-devel-centos7将对应的基础镜像拉下来
  • 新建空目录Docker, 下载对应的安装包, 放入Docker/envs/下
  • 使用tar -xvf cudnn-linux-x86_64-8.8.1.3_cuda12-archive.tar.xz解压得到cudnn-linux-x86_64-8.8.1.3_cuda12-archive
